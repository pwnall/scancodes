# scancodes

scancodes is a Go package that produces the Set 1 (IBM PC XT)
[keyboard scancodes](http://en.wikipedia.org/wiki/Scancode) necessary to type
in a string.

This is the author's secod piece of Go code, so feedback is welcome.
